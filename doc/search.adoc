The `search` command searches for a given pattern

USAGE:
    sshq search [OPTIONS] [PATTERN]

ARGS:
    <PATTERN>    Search pattern

OPTIONS:
    -h, --help       Print help information
    -j, --json       Output as json
    -V, --version    Print version information
